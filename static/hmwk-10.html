Counting Process Simulation

Single Realization

Total Count Distribution

Analysis

This simulation models a Homogeneous Poisson Process. The total time \(T\) is divided into \(n\) discrete intervals. Each interval is a Bernoulli trial. The probability of an event happening in any one interval is \(p = \lambda \frac{T}{n}\). It's possible to define \(dt\) = \(\frac{T}{n}\) so the formula became \(p = \lambda dt\)
Where:

  • \(\lambda\) is the average rate of occurrence.
  • \(T\) is the total time interval.
  • \(n\) is the number of subintervals.
The expected value is \(E[N(T)] = \lambda T\)
and the variance is \(Var(N(T)) = \lambda T\).
So the \(\lambda \) parameter directly influences both the mean and variance of the counting process.
